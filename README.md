# Library Management System

## Project Description
A simple Python-based Library Management System that helps manage book records. Users can add books, view available books, and search for books using their ID.

## Features
- Add new books
- View all books
- Search books by ID
- User-friendly menu-driven interface

## Technologies Used
- Python 3

## Project Structure

Library-Management-System/
│
├── library_management.py
├── README.md
└── requirements.txt

## How to Run

1. Clone the repository:
   git clone <repository-url>

2. Navigate to the project folder:
   cd Library-Management-System

3. Run the program:
   python library_management.py
